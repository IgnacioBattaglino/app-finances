# app-finances

Aplicación web de finanzas personales (PWA) para el seguimiento de gastos, ingresos e inversiones, con enfoque en la independencia financiera (FIRE).

Proyecto personal en desarrollo — sirve como herramienta de uso real y como pieza de portfolio.

## Stack
- React + Vite
- Tailwind CSS
- Supabase (PostgreSQL)
- PWA (instalable en móvil)
- Deploy en Vercel
